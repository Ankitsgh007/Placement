# Java Practice Problems

This repository contains Java practice exercises organized under `Amar-sir/` directory, covering:
- Basics (e.g., even/odd, sum series, swap numbers)
- Functions (e.g., factorial, palindrome, reverse number)
- 2D Arrays (e.g., sum, print, even/odd count)

## Structure
- `Amar-sir/Basics/`
- `Amar-sir/function/`
- `Amar-sir/2D Array/`
